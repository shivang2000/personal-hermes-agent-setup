## Notes on writing the row in `data/applications.md` (or any tracker)

The Notes column should be terse but searchable. Format:

```
{Date} | {Company} ({status}) | {Role} | N/A | BLOCKED | ❌ | [{NNN}](reports/{file}.md) | {Mitigation} {IP-level|fingerprint} block; tried {N} methods — see report. {Weak signal if any}. Needs {paste-JD | different-network | company-page}.
```

Status: `BLOCKED` (its own value, distinct from `Applied` / `Evaluated` / `SKIP`).

PDF column: `❌` (no PDF was generated because there was no JD to base it on).
