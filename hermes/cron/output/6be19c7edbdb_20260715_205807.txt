**Top picks**
- **Grok Build was exfiltrating user codebases to cloud storage** ([The Verge](https://www.theverge.com/tech/711847/spacexai-grok-build-codebase-leak)) — xAI's IDE-style "Grok Build" tool was uploading entire local codebases by default; Musk confirmed and promised deletion. Concrete lesson for anyone using coding agents: never trust implicit "telemetry" on agent tools, pin the exact files you send to model context, and treat agent IDEs like untrusted binaries.
- **Claude `web_fetch` exfiltration loophole via nested fetched links** ([Simon Willison](https://simonwillison.net/2026/Jul/15/claude-web-fetch-exfiltration/)) — Researcher Ayush Paul showed that Claude's `web_fetch` could follow links *inside* already-fetched pages, bypassing Anthropic's "only user-entered URLs" guard and leaking chat-memory data (name, city, employer) via a honeypot. Anthropic closed the gap by disallowing navigation to URLs returned *within* fetched content. The most concrete recent example of why the "lethal trifecta" (private data + untrusted fetch + exfil) keeps recurring in production agent designs.
- **Cloudflare Precursor: behavioral detection for agentic bots** ([Cloudflare blog](https://blog.cloudflare.com/precursor/)) — New continuous client-side validation engine that turns full-session user-journey signals (mouse, timing, focus) into bot-detection features, sitting behind Turnstile. Built for the world where headless agents, browser-use tools, and scrapers behave nothing like the old "one request, one script" model.
- **GitHub Copilot code review: why better tools made it worse, and the fix** ([GitHub Blog](https://github.blog/ai-and-ml/better-tools-made-copilot-code-review-worse/)) — Napalys Klicius explains how the review agent's "Unix-style code exploration" migration cut review cost by reshaping the agent's workflow around PR evidence instead of full repo walks. Practical architecture write-up for anyone building code-review or diff-understanding agents.
- **Cloudflare posts on the .AL DNSSEC rollover outage + EDE 33 signaling** ([Cloudflare blog](https://blog.cloudflare.com/al-dnssec-rollover/)) — When a failed DNSSEC key rollover took down Albania's TLD, 1.1.1.1 served a Negative Trust Anchor AND returned new EDE 33 so resolvers can see DNSSEC validation was bypassed.
