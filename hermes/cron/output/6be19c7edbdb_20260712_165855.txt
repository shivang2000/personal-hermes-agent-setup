**Top picks**
- [Mesh LLM distributes inference across heterogeneous machines](https://www.iroh.computer/blog/mesh-llm) — Iroh’s new runtime exposes an OpenAI-compatible endpoint while routing requests to peers or pipeline-splitting model layers across nodes. Authenticated QUIC, NAT traversal, gossip-based capability discovery, and relay fallback make this a notable attempt at practical decentralized inference.
- [GitHub: Better tools initially made Copilot code review worse](https://github.blog/ai-and-ml/github-copilot/better-tools-made-copilot-code-review-worse-heres-how-we-actually-improved-it/) — Giving the reviewer more tools increased autonomy but also tool misuse and low-value comments. GitHub’s production lesson is that agent quality depends on tool-selection training, trajectory-level evaluation, and end-to-end acceptance metrics—not simply expanding the action space.
- [ClickHouse scales PgBouncer to four times its previous throughput](https://clickhouse.com/blog/pgbouncer-clickhouse-managed-postgres) — The Managed Postgres team details its work around PgBouncer’s single-threaded event loop and connection-management bottlenecks. Useful reading for anyone operating high-connection-count Postgres services where the pooler, rather than the database, becomes the limiting component.
- [AI-assisted review finds an Ethereum consensus-layer vulnerability](https://www.coindesk.com/tech/2026/07/11/ethereum-foundation-says-ai-found-bug-that-could-take-validators-offline/) — The reported flaw could have forced validators offline, but human reviewers were still needed to validate exploitability and coordinate remediation. The important signal is AI moving from generic code scanning toward protocol-aware vulnerability discovery—not autonomous security sign-off.
- [Reame explores CPU-first LLM serving that improves over time](https://github.com/swellweb/reame) — This early-stage `llama.cpp`-based server combines persistent disk KV cache, generation archives, adaptive speculative decoding, and interleaved multi-user scheduling. It is an interesting design point for private inference on commodity hardware, though still too young for production assumptions.
- [CISA adds two actively exploited vulnerabilities to KEV](https://www.cisa.gov/news-events/alerts/2026/07/10/cisa-adds-two-known-exploited-vulnerabilities-catalog) — CVE-2026-48939 and CVE-2026-56291 now have evidence of exploitation in the wild. Teams should prioritize exposed affected assets and investigate for compromise rather than treating patch installation alone as sufficient remediation.

**Software Engineering & DevTools**
- [Buf introduces `protobuf-py`](https://buf.build/blog/protobuf-py) — The from-scratch Python Protobuf runtime covers proto2, proto3, editions, extensions, unknown fields, dynamic messages, and the conformance suite. Generated code is readable and typed with no runtime dependencies; an optional Rust accelerator targets performance comparable to Google’s `upb`.
- [Ant: a new JavaScript runtime and package ecosystem](https://antjs.org/) — The newly launched project is attempting an integrated runtime/tooling stack rather than another thin Node-compatible launcher. It is worth watching, but compatibility breadth, package installation security, and sustained maintenance will matter more than launch benchmarks.
- [Why SQLite applications should consider `STRICT` tables](https://evanhahn.com/prefer-strict-tables-in-sqlite/) — `STRICT` mode preserves SQLite’s lightweight deployment while rejecting values that cannot be losslessly coerced to declared column types. It is a low-cost guard against data-quality bugs for new schemas, especially those shared across several application languages.
- [Biff.graph models a Clojure codebase as a queryable graph](https://github.com/jacobobryant/biff/tree/v2.x/libs/graph) — Functions, dependencies, configuration, and application structure become graph data that tooling can query. The approach is especially relevant to code intelligence and agents, which benefit from explicit structural relationships rather than repeated text search.
